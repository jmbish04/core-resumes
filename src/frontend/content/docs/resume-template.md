# Resume Template

This page displays an embedded preview of your **Google Docs resume template**. The template is used by OrchestratorAgent when generating tailored resumes for job applications via the `create_doc_from_template()` method.

## How Templates Work

When you intake a new role, Colby copies this template into the role's Google Drive folder and replaces placeholder variables (like company name, job title) with role-specific values. You can then review and refine the generated resume.

## Template Configuration

The resume template ID is stored in the `template_ids` configuration under the key `resume`. You can update it on the [Config page](/config).
